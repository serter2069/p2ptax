# LANDING STRUCTURE — P2PTax

## Sections (in order)
1. **Hero** (above fold) — H1 + H2 + 2 CTA (создать заявку / найти специалиста) + visual
2. **Problem** — 3 карточки боли (страх / незнание / не найти нужного)
3. **Solution** — narrative: маркетплейс P2P-специалистов по ФНС
4. **How it works** — 3 шага (путь клиента: описать → получить предложения → выбрать)
5. **Features** — 4 карточки: фильтр по ФНС / специалисты пишут первыми / бесплатно / два пути входа
6. **CTA** — финальный призыв + повтор двух кнопок
7. **Footer** — ссылки, legal, контакты

## Navigation
- Sticky header: логотип + кнопка "Создать заявку" (primary CTA)
- Якорные ссылки: #how-it-works, #specialists, #faq (если добавим)
