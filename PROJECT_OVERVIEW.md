# P2PTax (Налоговик) — Project Overview

## Tagline
Налоговый информационный арбитраж

## Description
Маркетплейс для поиска налоговых специалистов (юристов, консультантов) в любом городе России. Клиенты публикуют заявки, специалисты откликаются и общаются через чат. Платформа включает онбординг специалистов и модерацию администраторами.

## Roles
- **CLIENT** — Ищет налогового специалиста. Публикует заявку, получает отклики, выбирает специалиста и общается в чате.
- **SPECIALIST** — Налоговый консультант или юрист. Проходит онбординг, видит ленту заявок, откликается, общается с клиентами.
- **ADMIN** — Модерирует специалистов, заявки и отзывы. Управляет пользователями и промо-тарифами.
- **GUEST** — Незарегистрированный пользователь. Просматривает лендинг, каталог специалистов, ленту заявок. Может создать заявку без регистрации.

## Scenarios
### S-001: Клиент создаёт заявку (CLIENT)
1. Открывает дашборд, нажимает "Создать заявку"
2. Заполняет форму: описание, город, ФНС, тип услуги
3. Заявка создана, ожидает откликов
4. Просматривает отклики специалистов
5. Начинает переписку с выбранным специалистом

### S-002: Гость — быстрая заявка с лендинга (GUEST)
1. Видит форму быстрой заявки, вводит описание + email
2. Вводит OTP-код из письма (dev: 000000)
3. Аккаунт создан, заявка опубликована

### S-003: Регистрация специалиста (SPECIALIST)
1. Вводит email, выбирает роль "Специалист"
2. Вводит OTP-код
3. Выбирает уникальное имя
4. Указывает город, ФНС, виды услуг
5. Загружает фото, пишет о себе
6. Онбординг завершён, кабинет открыт

### S-004: Специалист откликается (SPECIALIST)
1. Видит новые заявки в дашборде
2. Читает детали заявки
3. Пишет сообщение клиенту, указывает цену и срок
4. Отклик отправлен, ожидает ответа клиента

### S-005: Модерация профиля специалиста (ADMIN)
1. Видит профили в очереди модерации
2. Проверяет документы и данные
3. Одобряет или отклоняет профиль с комментарием
4. Специалист становится видимым в каталоге
